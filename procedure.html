DNA sequence analysis and determining DNA open reading frames

Procedure to Work Simulator

    1. Provide a DNA sequence in user interface for finding ORF. For example provide a hypothetical sequence CGCTACGTCTTACGCTGGAGCTCTCATGGATCGGTTCGGTAGGGCTCGATCACATCGCTAGCCAT

    1. Click on Find ORF tab for executing simulator.
    1. A default sequence file is provided in the user interface. User can download the file and can use the file as query sequence for finding ORF.

ORF is a continuous stretch of codons with a start codon usually AUG and with a stop codons usually UAA, UAG or UGA.
